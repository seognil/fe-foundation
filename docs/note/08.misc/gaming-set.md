### （笔记）

#### 4K

真 4K == True 4K == 抖 4K

原生 4K == 不是抖出来的

万元价位的 4K 都是抖出来的，原生 4K 真买不起…

#### 对角线英寸 -> 厘米宽度 系数：2.2138

120 英寸 == 宽 266 厘米

```ts
2.54 * (16 / (16 ** 2 + 9 ** 2) ** 0.5); // => 2.213801864296355
```

#### 投射比：投影距离 / 成像宽度

投射比越小，需要的投屏空间越小

比如 `RS Pro2` （宣传页） 1.2 的投射比：

```ts
1.2 * (150 * 2.2138); // => 3.98 米
```
