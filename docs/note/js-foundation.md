## 大致路线图

对于**学习 JavaScript 语言**来说，我认为合适的方式大致是：

- 准备：预先理解最基本的编程思想
- 入门：熟悉 JavaScript 基本语法，了解 JavaScript 的大致样貌
- 熟悉：练习 JavaScript 常用内置对象的常用 API 用法
- 实战：熟悉 JavaScript 能够进行的进行实际开发
- 深入：了解 JavaScript 深层次的语言特性和原理
